**用途**：VI 升级的战略输入文档 · 给视觉设计方的任务单

**期望返回**：W2 初稿 · W2 末视觉拍板

## 1. 背景与目标 · Why

### 1.1 公司身份定位
- **港股上市公司**（2415.HK），医学专业服务集团
- **3.33M+ 注册医生网络**，覆盖中国高级医生
- **AI 转型第二年**：业务核心在 AI-Enabled Delivery + PITL（Physician-in-the-Loop）
- **双主线业务**：Entering China（全球企业入华）+ Going Global（中国创新者赴美）

### 1.2 VI 升级的三个必须回答

1. **"AI 时代"**：视觉必须传达 AI 原生、前沿、技术驱动，但不能脱离医学严肃性
2. **"上市公司"**：视觉必须承载 2415.HK 身份的可信度与稳重，反向尽调时不输美股医疗 SaaS 同行
3. **"跨境可信"**：美国客户/投资者与中国客户/医生网络同时能感觉到"专业、对得上"

### 1.3 VI 升级不是做什么

- 不是 Logo 重做（Logo 保留）
- 不是推倒重来（品牌资产延续）
- 不是追潮流（这一代 VI 需要支撑 3–5 年）

## 2. 品牌关键词（Keywords）

### 2.1 要承载的调性（Must-Have）

| 关键词 | 含义 | 视觉暗示 |
|---|---|---|
| Trust · 可信 | 上市公司级别、医学级严谨 | 克制的色彩、充足留白、严谨字体 |
| Clinical · 临床专业 | 医生、证据、出版物 | 干净版式、精确排版、医学图像 |
| Cross-Border · 跨境 | 中美双向、非"中式"非"美式" | 国际化克制视觉、避免中国红或星条旗符号 |
| AI-Native · AI 原生 | 技术驱动、数据前置 | 几何造型、数据可视化痕迹、动态网格 |
| Human-in-the-Loop · 人机协同 | 医生审阅保证质量 | 人像元素节制、医学插画点缀 |

### 2.2 要避免的调性（Must-Avoid）

| 反面调性 | 原因 |
|---|---|
| 过度 tech bro / 硅谷初创感 | 会削弱上市公司可信度 |
| 过度保守 / 企业门户 1998 | 不支撑 AI 转型叙事 |
| 中药 / 保健品 / 养生品 | 战略上主站不承载国内消费医疗 |
| 星条旗 / 中国红 / 地球仪 | 过度直白的跨境符号 = 低级国际化 |
| 医院蓝 / 十字 / 听诊器 | 过度直接的医疗符号 = 低端医疗门户 |
| 渐变光斑 / 玻璃拟物 / Web 3 感 | 已过时；反向尽调扣分 |

## 4. 色彩方向建议（Color Direction）

### 4.1 候选色系 A · 克制深蓝 + 科学绿（推荐 ⭐）

```
Primary:    #0A2540 (深海军蓝) — 权威、稳重、可信
Secondary:  #1E6F5C (深森林绿) — 临床、科学、成长
Accent:     #00C6B8 (鲜青) — AI 前沿、能量点
Neutrals:   #0F172A / #334155 / #64748B / #CBD5E1 / #F8FAFC
Surface:    #FFFFFF / #F8FAFC（正文背景以白为主）
Warning:    #F59E0B · Error: #DC2626 · Success: #10B981
```

**心理语言**：上市公司级稳重 + 临床专业 + 一丝 AI 前沿
**落地提示**：Primary 用于导航、CTA、标题；Accent 仅用于 5% 面积的数据点缀、Hover 态、数字强调

### 4.2 候选色系 B · 石墨灰 + 暖金（次选）

```
Primary:    #111827 (石墨黑)
Secondary:  #B8860B (暖金，用于强调而非奢华)
Neutrals:   冷灰阶
```

**心理语言**：更偏研究机构 / 学术出版物的调性
**适合场景**：若 Sponsor 倾向"学术期刊级"严肃感可选，但对 AI-Native 信号略弱

### 4.3 候选色系 C · 双主色（不推荐）

Entering China 一个色，Going Global 一个色 —— 在不同页面强调不同主色。
**不推荐原因**：v4.1 §1.3 明确双主线视觉等权重，双主色会弱化任一侧。

## 5. 字体方向建议（Typography Direction）

### 5.1 英文字体（EN）

| 位置 | 推荐 | 理由 |
|---|---|---|
| Display / Hero | **Inter Display** 或 **Söhne** | 中性、上市公司级、不流俗 |
| Body | **Inter** | 屏幕可读性最佳、开源、授权可商用 |
| Mono（代码 / 数字） | **JetBrains Mono** | 技术感点缀 |

**备用方案**：若预算允许授权商业字体 → Söhne（Klim Type Foundry）
**开源首选**：Inter（全免费）

### 5.2 中文字体（CN）

| 位置 | 推荐 | 理由 |
|---|---|---|
| Display / Hero | **思源黑体 Heavy / Bold**（Source Han Sans） | 开源、与 Inter 匹配度高 |
| Body | **思源黑体 Regular / Medium** | — |

## 7. 图像 / 摄影 / 插画风格

### 7.1 摄影（Photography）

**主方向**：克制纪实医学影像
- 真实临床场景（实验室、会议、医生操作），但**不过度摆拍**
- 自然光为主，暖冷中性，避免过度滤镜
- 人像不使用刻板"白大褂 + 微笑"stock 照

**禁用**：
- 图库感 stock 照（所有用过的都不能用）
- AI 生成的合成医疗图（FTC 合规风险）
- 过度美颜 / 戏剧化打光

### 7.2 数据可视化

**主方向**：严谨、克制、信息密度适中
- 使用 Primary + Accent 双色，避免彩虹渐变
- 标题 + 数值 + 单位 + 年份 + 来源，五要素必备
- 动画节制（≤ 400ms），避免 "fancy" 图表

### 7.3 插画（Illustration）

**主方向**：几何抽象 · 低饱和
- 用于流程示意、能力框架、跨境场景
- 风格：线条 + 平面色块，避免 3D 拟物与卡通化
- 参考：Stripe、Linear 的功能插画密度

**禁用**：
- 过度 AI 生成感（蓝紫渐变 + 神经网络图腾）
- 卡通化人物
- 国风水墨 / 剪纸

## 8. 图标与组件（Iconography & Components）

### 8.1 图标库选择

**首选**：Lucide Icons（开源、与 Tailwind 生态契合、stroke 风格克制）
**参数**：stroke 1.5px · size 16/20/24 · 圆角 2px

### 8.2 关键组件视觉语言

- **Button Primary**：Primary 色填充 · 字体 Semibold · 圆角 6px · Hover 位移 -1px + Accent 边框（1px）
- **Button Secondary**：透明填充 · Primary 边框 1px · Hover 背景 Neutral-50
- **Card**：白底 · 边框 Neutral-200 (1px) · 圆角 12px · 阴影仅在 Hover 出现
- **Input**：边框 Neutral-300 · Focus Accent 2px 外边光
- **Data Badge**（用于数据 claim 年份标注）：小号灰字 + hover 显示来源 tooltip

## 9. 声音与调性（Voice & Tone）

Copy Deck v4.1 已基本定调，此处补充视觉层的呼应：

| 原则 | 视觉对应 |
|---|---|
| 诚实边界 · 不夸大 | 不用"Industry-leading" badge、不用"Trusted by 1000+"的大字墙 |
| 证据驱动 | 所有数字都带年份 + 来源（可 hover tooltip） |
| 克制专业 | 版式留白充足、CTA 不超过 2 个层级、动效节制 |
| 医学严谨 | 不用"魔法"/"突破"/"颠覆"这类文案，视觉上对应不用爆炸/闪光/奖杯 icon |

## 11. 评审决策机制

### 11.1 拍板权

- **视觉方向**（色系 / 字体 / 风格）：市场负责人决策，Sponsor 批准
- **组件细节**：Designer 自主决策
- **关键页面视觉稿**（首页 / 双主线）：市场 + Tech Lead + Sponsor 三方签字
- **合规相关视觉**（数据 claim 展示、logo 墙）：法务需签字

### 11.2 评审节奏

- **每日 15min stand-up**（仅 Designer + PM + 市场）：问题早发现
- **周评审 30min**（全员 optional）：方向修正
- **关键节点正式拍板会**：M1 视觉拍板 / M3 双主线页拍板

### 11.3 修改限制

每份视觉稿最多 **2 轮修改**；第 3 轮需 Sponsor 重新入场说明理由，否则 Designer 方向自主权不可侵蚀。
