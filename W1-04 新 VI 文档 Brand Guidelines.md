## 3. 字体系统 · Typography

### 3.1 字体家族（Font Stack）

```css
/* Display · 用于 H1 / Hero（呼应 Logo 衬线字标） */
font-display: "Source Serif 4", "Source Serif Pro", "Noto Serif SC",
              Georgia, "Source Han Serif SC", serif;

/* UI · 正文 / 按钮 / 导航 */
font-ui:      "Inter", system-ui, -apple-system, "Helvetica Neue",
              "思源黑体", "Source Han Sans SC", sans-serif;

/* Slogan · 主 Slogan 专用（Italic 模拟 Logo tagline） */
font-slogan:  "Source Serif 4", Georgia, serif;
font-style:   italic;

/* Mono · 代码、规范数据 */
font-mono:    "JetBrains Mono", "SF Mono", Menlo, Consolas, monospace;
```

**为何引入 Source Serif 4**
- Logo 字标为 Transitional Serif；Source Serif 4 是 Adobe 开源 Transitional Serif，骨架最接近
- 开源、Google Fonts 托管，无授权风险
- Variable font（支持 8–60 光学尺寸），大字锋利、小字平稳
- 与 Inter 搭配在医学 / 科研 / 咨询类网站（Veeva、IQVIA 参考）是主流组合

### 3.2 字阶（Type Scale · 1.25 Major Third）

| Token | Size | Line-height | Letter-spacing | Weight | **Family** | 用途 |
|---|---|---|---|---|---|---|
| `display-xl` | 72px | 1.1 | -0.015em | 600 | **Serif** | Hero Headline |
| `display-l` | 56px | 1.15 | -0.012em | 600 | **Serif** | Hero Subheadline / Section Hero |
| `h1` | 40px | 1.2 | -0.01em | 600 | **Serif** | Page Title |
| `h2` | 32px | 1.25 | -0.005em | 700 | Sans (Inter) | Section Title |
| `h3` | 24px | 1.35 | 0 | 600 | Sans | Block Title |
| `h4` | 20px | 1.4 | 0 | 600 | Sans | Card Title |
| `body-l` | 18px | 1.6 | 0 | 400 | Sans | Lead paragraph |
| `body` | 16px | 1.6 | 0 | 400 | Sans | 正文 |
| `body-s` | 14px | 1.5 | 0 | 400 | Sans | 注释 / 表单 |
| `caption` | 12px | 1.4 | 0.03em | 500 | Sans | 脚注 / 来源标注 |
| `slogan-main` | 28px | 1.3 | 0 | 400 italic | **Serif Italic** | 主 Slogan *Improving Healthcare Quality* |
| `slogan-sub` | 18px | 1.4 | 0 | 500 | Sans | 子 Slogan |

### 3.3 字重使用规则

- Regular (400)：正文 / 主 Slogan Italic
- Medium (500)：表单、菜单项、tag、子 Slogan
- Semibold (600)：Serif H1/Display、Sans H3/H4、按钮、强调
- Bold (700)：Sans H2
- 避免 Extrabold / Black（过于厚重，不符合医学克制气质）
- **禁止将 Source Serif Bold 700 用于正文长段**（阅读疲劳）

### 3.4 中英文混排规则

- 中文用思源黑体 / 思源宋体；英文 UI 用 Inter、英文 Display 用 Source Serif 4；浏览器 fallback 自动处理
- 段落内中英混排，英文前后插入 `<wbr>` 保证换行合理
- 数字永远使用 Inter（不使用思源黑体的数字、不使用 Source Serif 的数字跨栏）
- CN 白名单页面（Homepage / About / Legal / IR）的 H1 使用 **思源宋体 SemiBold**（与 Source Serif 同为衬线族）

## 4. 网格与布局 · Grid & Layout

### 4.1 容器最大宽度
- Desktop：`max-width: 1280px`（可选 1440px 大屏变体）
- 内容排版：`max-width: 960px` 保证正文可读性
- Hero：可突破到 `100vw`，内部内容仍约束在 1280px

### 4.2 栅格

| 断点 | 列数 | Gutter | 边距 |
|---|---|---|---|
| `xs` (≤ 640px) | 4 | 16px | 20px |
| `sm` (641–768px) | 6 | 20px | 24px |
| `md` (769–1024px) | 8 | 24px | 40px |
| `lg` (≥ 1025px) | 12 | 24px | 80px |

### 4.3 间距 token（8px 基线）

```
space-1   = 4px       space-10  = 40px
space-2   = 8px       space-12  = 48px
space-3   = 12px      space-16  = 64px
space-4   = 16px      space-20  = 80px
space-5   = 20px      space-24  = 96px
space-6   = 24px      space-32  = 128px
space-8   = 32px
```

Section 之间纵向呼吸：Desktop `space-24` ~ `space-32`，Mobile `space-16` ~ `space-20`。

## 8. 图像与插画 · Imagery

### 8.1 图标库
- **Lucide Icons**（开源，stroke 1.5px，sizes 16/20/24）
- 禁用多色图标（保持单色克制）
- 强调用 Primary；普通用 Neutral-700；AI/数据场景可用 Accent-700

### 8.2 照片风格

- 真实临床 / 会议 / 工作场景
- 自然光偏冷中性
- 禁用图库 stock photo
- AI 生成图禁用于医学内容（FTC 合规）

### 8.3 插画

- 几何抽象 + 平面色块
- 仅使用 Primary + Accent + Neutral，**不使用 Success 绿**
- 线宽 1.5–2px，与图标 stroke 同步
- **推荐母题**：承接 Logo 的"点阵螺旋"——用不同尺寸蓝色圆点构成数据 / 网络 / 流程的抽象视觉

### 8.4 数据可视化

- 主色 Primary-700，辅助 Primary-500，强调 Accent-500
- Success 仅用于"验证通过"类数据标签（功能语义）
- 标签字体 caption，数字 Inter Semibold
- 必带标题 + 单位 + 年份 + 来源
- 动效限制：渐入 ≤ 400ms，禁用无限循环动画

## 9. 动效与声音 · Motion & Voice

### 9.1 动效原则

- 动效服务于引导，不服务于炫技
- 全局 transition：`200ms`（UI 反馈）/ `400ms`（内容进入）
- Easing：`cubic-bezier(0.4, 0, 0.2, 1)`
- 必须支持 `prefers-reduced-motion: reduce`

### 9.2 禁忌

无限循环动画、Parallax 多层滚动、光标跟随动效、超 600ms transition。

### 9.3 文案总则（承 Copy Deck v4.1）

- 诚实边界，不夸大
- 证据驱动，带源带年
- 第二人称（You / Your team）
- 英文为主，医学专业术语优先学术写法

### 9.4 禁用词（合规红线）

同 Copy Deck v4.1 §6.3：
- zero-hallucination / hallucination-free
- 100% accurate / perfect accuracy / flawless
- industry-leading / best-in-class / world's #1
- 任何无年份无来源的数据
- 挪用集团 / 兄弟公司案例冒充 MedSci Healthcare 直接交付案例

### 9.5 首选表述替换

| 避免 | 推荐 |
|---|---|
| industry-leading | recognized / proven |
| cutting-edge AI | AI-assisted / AI-enabled |
| revolutionary | more structured / more defensible |
| trusted by thousands | engaged with healthcare teams across ... |

### 9.6 Slogan 本地化

| 英文 | 中文 |
|---|---|
| Improving Healthcare Quality | 提升医疗质量（与 Logo 关联时不翻译，保留英文） |
| AI-Enabled. Physician-Verified. Globally Ready. | AI 驱动 · 医生把关 · 全球就绪 |

## 10. 响应式与无障碍 · Responsive & A11y

### 10.1 断点

```
xs  ≤ 640px     (mobile)
sm  641–768px   (large mobile)
md  769–1024px  (tablet)
lg  1025–1440px (desktop)
xl  ≥ 1441px    (large desktop)
```

### 10.2 Mobile-First 原则

- 移动端首屏仅 1 个 CTA（推荐 Entering China 或根据 UTM 动态切换）
- 导航汉堡化后垂直单列；Logo 切换为 Mark
- Case Study 3 metrics 从横向变纵向 stack
- Hero 标题字号自动从 `display-xl` 降到 `display-l` 或 `h1`

### 10.3 a11y（WCAG 2.1 AA）

- 正文对比度 ≥ 4.5:1
- 大字 + UI 组件 ≥ 3:1
- 所有交互元素键盘可达
- Focus 状态 Accent 2px 外环
- alt 文本全量；Logo alt 为 "MedSci Healthcare — Improving Healthcare Quality"
- 表单 label 关联
- 支持 `prefers-reduced-motion`
